# java-cli-maven-drool-car-data

## Description
A POC for using drools framework.

## Tech stack
- java
- maven
  - drools

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`

## Credit
[Code based on](https://www.codeusingjava.com/drools/stateless)
